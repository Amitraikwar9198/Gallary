✨ About Images

Welcome to Gallery — a visually immersive, fully responsive image gallery that transforms browsing into an experience.

Every image is presented with elegance using smooth transitions, interactive hover overlays, category-based filtering, and full-screen lightbox views — creating a gallery that feels alive and intuitive to navigate.

Whether you're an artist, a designer, a developer, or simply an admirer of beautiful visuals, this gallery offers an inspiring and technically refined showcase.

🔧 Features You’ll Love

  • 📁 Category-based filtering (Nature, Animals, Architecture)
  • 💡 Hover overlays with dynamic captions
  • 🖼️ Lightbox viewer with next/prev navigation
  • ⚡ Optimized for performance and responsiveness
  • 🎨 Clean, minimalist, modern design principles

🛠️ Built With

HTML5, CSS3 (Flexbox, Grid, Transitions), Vanilla JavaScript (DOM, Events), and Responsive Web Design techniques. External image APIs like picsum.photos make each visit unique.
